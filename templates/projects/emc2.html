E=MC2 (ve çözümü)

Problemi yazan: Chris, keşfeden: Einstein

İnsanlığın heyecan verici keşiflerinden biri de kütle ve enerjinin birbirine dönüşebiliyor olmasıydı. Ve aralarındaki ilişki meşhur denklem E = M × C 2 ile ortaya koyuldu. Bu ilişki neredeyse 100 yıl önce Albert Einstein tarafından ortaya çıkarıldı ve şu an biliyoruz ki güneş küçük miktarda kütleleri büyük miktarlarda enerjiye dönüştürerek ısısını üretiyor.

Kullanıcıdan sürekli olarak kütleyi okuyan ve enerji miktarını gösteren bir program yazınız. Hesaplamanızı aşağıdaki şekilde gösteriniz:

Eğer kullanıcıdan kütleyi kilogram biriminde alırsanız, ve ışık hızını 299792458 (metre/saniye) alırsanız, E = M × C 2 hesabının sonucunu Joule cinsinden bulursunuz.

3.0E8 , 3.0 × 10 8 ve 300.000.000'un bilimsel gösterimidir, bunu dikkate alın. Java'da double'ları bilimsel gösterimle göstermek için bir şey yapmanıza gerek yok çünkü sayılar yeterince büyük olduğunda otomatik olarak öyle gösterecek.

Işığın hızı hiç değişmediği için, onu sabit (constant) olarak tanımlamak mantıklı.

Çözüm

/**
 * Program: EMC2
 * -------------
 * This program helps users calculate how much energy they could
 * get if they transformed their mass. Thanks Einstein!
 */
public class EMC2 extends ConsoleProgram {
	
	// This declares a constant... it can never change value.
	// Speed of light in m/s
	private static final double C = 299792458;
	
	public void run() {
		while(true) {
			// Read the mass in from the user.
			double massInKg = readDouble("Enter kilos of mass: ");
			
			// Calculate energy
			double energyInJoules = massInKg * C * C;
			
			// Display work to the user
			println("e = m * C^2...");
			println("m = " + massInKg + " kg");
			println("C = " + C + " m/s");
			println(energyInJoules + " joules of energy!");
			println("");
		}
	}
}
