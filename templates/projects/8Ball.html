8 Top (ve çözümü)

Problemi yazan: Chris Piech.

8 Top'un arkasındaki fikir çok basit. Topa bir evet hayır sorusu soruyorsunuz, ve o da size cevabı söylüyor.

Cevabın önceden hazırlanmış cevaplardan rastgele seçiliyor olması haricinde doğru cevabı söylüyor.

Gerçek hayattaki bir 8 Top

Sürekli olarak kullanıcıdan bir evet hayır sorusu bekleyen ve sonrasında da önceden hazırlanmış 5 cevaptan birini rastgele olarak seçen bir progra yazın:

  • Şüphesiz.
  • Evet.
  • Sonra tekrar sorun.
  • Hayır.
  • Karel öyle düşünüyor.

Programın örnek bir çıktısını burada görebilirsiniz.

Rastgele sayılar üretmek için, şu şekilde bir RandomGenerator örnek değişkeni (tüm metotların dışında tanımlanmış) yaratın :

private RandomGenerator rgen = new RandomGenerator();

Sonrasında rastgele sayılar üretmek için bu değişkeni kullanabilirsiniz.

rgen.nextInt(max) //[0, max) aralığında rastgele bir int üretir

Çözüm

/**
 * Class: Eight Ball
 * -----------------
 * Simulates an eight ball and gives sage answers to yes or
 * no questions.
 */
public class EightBall extends ConsoleProgram {
	
	private static RandomGenerator rg = new RandomGenerator();
	
	public void run() {
		while(true) {
			readLine("Ask a yes or no question: ");
			
			int choice = rg.nextInt(5);
			println(choice);
			if(choice == 0) {
				println("Without a doubt");
			}
			if(choice == 1) {
				println("Yes");
			}
			if(choice == 2) {
				println("Ask again later.");
			}
			if(choice == 3) {
				println("No");
			}
			if(choice == 4) {
				println("Karel thinks so.");
			}
			println("");
		}
	}
}
